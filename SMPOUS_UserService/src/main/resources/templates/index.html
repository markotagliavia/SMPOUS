Microservices Demo - User Representation

Example

  • Simple message - directly:http://localhost:2222/simpleMessage
  • Simple message - regular:/simpleMessageRepresentation
  • Send message - directly:http://localhost:2222/hello/probnaPoruka
  • Send message - regular:/sendHello/probnaPoruka
  • Simple message - router:http://localhost:8765/user-service/simpleMessage
  • Send message - router:http://localhost:8765/user-service/hello/probnaPorukaDva
  • Simple message - router - serviceId:http://localhost:8765/mcs-user-service/simpleMessage
  • Send message - router - serviceId:http://localhost:8765/mcs-user-service/hello/probnaPorukaDva

Spring Boot URLs

For those interested, Spring Boot provides RESTful URLs for interrogating your application (they return JSON format data):

  • The beans
  • The environment
  • Thread dump
  • Application health
  • Application information
  • Application metrics
  • Request call trace
